# Firmware update
1. Download and install nRF Device Firmware Update from playmarket
2. Download .zip package from release page of this repo
3. Start nRF Device Firmware Update
4. Select firmware .zip package<br>
![img](images/select_file.jpg)
5. Select device. Device bluetooth name will be "Baklan Vario". If last attempt of firmware update goes wrong, press and hold power button at vario **till the end of firmware update** and search for "Baklan_dfu"<br>
![img](images/select_device.jpg)
6. Press and hold power button at vario **till the end of firmware update** and press "Start" button<br>
![img](images/start_update.jpg)
7. Wait for firmware update
